# Meltdown

Meltdown 是一个 Fusion 360 插件，提供一个面向建模工作的 AI 聊天面板。你可以直接在 Fusion 360 里用自然语言描述需求，让它帮你查看当前设计、规划建模步骤，并执行一部分建模操作。

目前支持的 AI provider：

- Gemini
- Claude
- OpenAI

## 安装

1. 下载或克隆这个仓库。
2. 将仓库中的 `meltdown` 文件夹放到 Fusion 360 的 Add-Ins 目录中。
3. 打开 Fusion 360。
4. 打开 `Scripts and Add-Ins`。
5. 找到 `meltdown`，点击运行。
6. 启动后如果聊天面板没有自动弹出，就点击工具栏中的 `Meltdown: Chat`。

## 首次启动

- 第一次启动时，插件会自动检查并安装依赖。
- 插件会优先尝试使用 `uv` 安装依赖。
- 如果系统里没有 `uv`，会退回到 `python3 -m pip`。
- 首次启动可能会慢一些，属于正常现象。
- 安装依赖和调用模型都需要网络连接。

## 基础配置

1. 打开 Meltdown 聊天面板。
2. 在输入框中输入 `/config`。
3. 在设置面板中选择要使用的 AI provider。
4. 填入对应的 model 名称。
5. 填入该 provider 对应的 API key。
6. 点击保存。
7. 返回聊天界面后就可以开始使用。

## 常用命令

- `/config`：打开设置面板，配置 provider、model 和 API key。
- `/clear`：清空当前对话历史。
- `/stop`：停止当前正在执行的操作。
- `/debug`：打开或关闭调试日志面板。
- `/history`：查看已保存的历史会话。
- `/resume`：从历史会话中恢复，继续之前的聊天。

## 使用示例

你可以直接输入自然语言，例如：

- `帮我创建一个 120mm x 80mm x 6mm 的安装板`
- `在四个角上各加一个直径 5mm 的通孔`
- `查看一下当前模型，告诉我现在有哪些 body 和 sketch`
- `帮我检查这个零件还缺哪些适合 CNC 加工的特征`

## 注意事项

- 这是一个 Fusion 360 内的聊天式工具，最适合用来做建模辅助和迭代操作。
- 如果当前操作执行时间较长，可以用 `/stop` 中断。
- 如果你想继续之前的工作，可以用 `/history` 或 `/resume` 找回历史会话。
- 如果你希望每次打开 Fusion 360 都自动加载它，可以在 `Scripts and Add-Ins` 里为这个插件开启启动项。
